My 100 Apps challenge (Why and How?)

2017-06-11

After having finished #100DaysOfCode, I realized that coding everyday became my habit. You might not know what is #100DaysOfCode.

This is a challenge for 100days to keep coding 1 hours at least everyday. This challenge was started by Mr.Kallaway and it became popular among coding learners. I finished my challenge in the end of May and I loved it!

I was thinking to start another #100DaysOfCode challenge, but I wanted to try something new on the other hand.

Then I came up with “100 Apps” ! As you can guess, this challenge is simple.
The goal is to make 100 Apps.

The definition

To begin with, “What is App?”. App is abbreviation of “Application”. Application contains several meanings. In terms of computing , the definition is as the below.

a program or piece of software designed to fulfill a particular purpose.

Programing is a way of problem solving. App is a program to solve these problems.

The purpose
I found out that building projects helps me to learn programming practically. Although it might not be effectively, as I was stuck so many times and spending tons of time to find out the solution.

The purpose of this challenge is to learn programing practically and build your own projects/portfolios.

The rule

Here is the rule. The rule is also simple.

1 . Coding at least 1 hour per day for App

Keep updating in the GitHub repository.
Share each App in Social media (and try to get feedback)
I don’t set up time frame for this challenge. At first, I was planning to make 100Apps in 100days.

I found out that it did not work . It’s easy to make very simple App but it takes time to make complicated App. I do need to build the later one. The purpose is learn new knowledge and skills through this challenge. :)
The challenge has started!

My 100 Apps challenge has started two weeks ago and I have built 7 Apps so far. (Some of them is not technically App.. I didn’t define the rule till now. I will focus on ‘Technically App’ from now on) .

My GitHub repository for 100Apps can find here. If you are interested in this challenge, why don’t you join me and start your own 100 Apps challenge?
